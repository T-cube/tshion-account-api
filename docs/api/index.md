## 基本信息

### 访问方式

访问地址：http://{server_address}:3000/api/{api_function}/{api_action}

支持动作：GET, POST, PUT, DELETE

### 数据格式

输入、输出数据格式均为JSON

#### 请求格式

POST、PATCH 请求头请加入：
```http
Content-Type: application/json
```

使用 Bearer 方式

```http
Authorization: Bearer {access_token}
```

#### 返回格式

```http
HTTP/1.1 200 OK
Content-Type: application/json

{
  ...
}
```

##### 错误

HTTP 错误编码通常为404, 400, 500

```http
HTTP/1.1 400 Bad Request
Content-Type: application/json

{
  "errno":{errno},
  "errmsg":"{errmsg}",
  "data":{result}
}
```
